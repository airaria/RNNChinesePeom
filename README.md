# RNNChinesePoem

A RNN-based LM model trained on quan_tang_shi.txt dataset, which can be used to generate poems.

The ckpt files are saved to save_model/.

the tensorboard's graph files are saved to graphs/.

## Requirements
- python 3.X
- Tensorflow >= 1.2
